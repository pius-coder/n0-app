
# _n0 Design System Tokens

## Colors
- Primary: blue-600 (CTA, links, accents)
- Text: neutral-900 (headings), neutral-600 (body), neutral-400 (muted)
- Backgrounds: white (default), neutral-50 (alternating sections), neutral-950 (dark sections)
- Success: green-500 | Warning: yellow-500 | Error: red-500
- Brand gradient: from-blue-600 to-violet-600

## Typography
- Hero: text-5xl sm:text-6xl lg:text-7xl font-bold tracking-tight
- Section title: text-3xl sm:text-4xl font-bold
- Section subtitle: text-lg text-neutral-600 max-w-2xl
- Body: text-base text-neutral-600 leading-relaxed
- Small: text-sm text-neutral-500
- Mono (brand): font-mono (for _n0, phone numbers)

## Spacing
- Section padding: py-20 sm:py-28 lg:py-32
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Section gap: space-y-16
- Grid gap: gap-6 sm:gap-8
- Element gap: gap-4

## Components
- Cards: rounded-2xl border border-neutral-200 bg-white shadow-sm hover:shadow-md transition-shadow
- Buttons: see ui/button variants
- Badges: rounded-full px-3 py-1 text-xs font-medium
- Dividers: border-t border-neutral-100
